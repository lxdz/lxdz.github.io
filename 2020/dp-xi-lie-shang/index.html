一 线性DP

经典dp三题，最长上升子序列，最长公共子序列，数字三角形

1.最长上升子序列

题目

给定一个长度为N的数列，求数值严格单调递增的子序列的长度最长是多少。

输入输出

输入:

7
3 1 2 1 8 5 6

输出:

4

数据范围

1≤N≤10001≤N≤1000，
−10^9≤数列中的数≤10^9

DP解法

思路

1.状态表示：dp[i]表示以第i个数结尾的数列，单调递增的子序列长度。
2.状态计算，j∈(0,1,2,..,i-1), 在w[i] > w[j]时，dp[i] = max(dp[i],dp[j]+1);
3.初始化:每个递增子序列都最少包含自身，所以dp数组每个数都应该初始化为1
4.目标值:结果是求所有序列的最大值min

复杂度分析

O(n^2)（状态数*转移数）

代码

//dp解法

#include<iostream>

using namespace std;

const int N=1010;

int f[N],dp[N];

int main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
        cin>>f[i];
    }
    for(int i=1;i<=n;i++){
        dp[i] = 1;
        for(int j=1;j<i;j++){
            if(f[i]>f[j]){
                dp[i] = max(dp[i],dp[j]+1);
            }
        }
    }
    int res = 0;    
    for(int i=1;i<=n;i++) res = max(res,dp[i]);
    cout<<res;
}

二分解法

思路

类似 patience game 纸牌游戏。

把点数小的牌压到点数比它大的牌上。如果当前牌点数较大没有可以放置的堆，则新建一个堆，把这张牌放进去。如果当前牌有多个堆可供选择，则选择最左边的堆放置。
把这个思路写成代码就是了。

复杂度分析

O(nlogn)

#include<iostream>

using namespace std;

const int N=1010;

int nums[N],f[N];
int cnt;

int main(){
    int n;
    cin>>n;
    for(int i=0;i<n;i++){
        cin>>nums[i];
    }
    f[cnt++] = nums[0];

    for(int i=1;i<n;i++){
        if(nums[i]>f[cnt-1]) f[cnt++] = nums[i]; //新开一个堆
        else{
            int l = 0,r = cnt-1;
            while(l<r){
                int mid = (l+r) >>1;//[l,mid][mid+1,r]
                if(f[mid]>=nums[i]) r = mid;
                else l = mid+1;
            }
            f[l] = nums[i];
        }
    }
    cout<<cnt;
}

2.最长公共子序列

题目

给定两个长度分别为N和M的字符串A和B，求既是A的子序列又是B的子序列的字符串长度最长是多少。

输入输出

输入

4 5
acbd
abedc

输出

3

思路

1.状态表示 dp[i][j] 表示串A[0-i] 串B[0-j] 的最长公共子序列长度
2.状态计算 如果A[i] == B[j] dp[i][j] = max(dp[i-1][j],dp[i][j-1],dp[i-1][j-1]+1);
3.初始化 dp[i][0] = 0,dp[0][j] = 0
4.目标值 dp[n][m],n,m表示A,B的长度

代码

#include<iostream>
#include<algorithm>
#include<string>

using namespace std;
const int N=1010;
int dp[N][N];
string A,B;

int main(){
    int n,m;
    cin>>n>>m;
    cin>>A>>B;
    A = ' ' + A;
    B = ' ' + B;

    for(int i=1;i<=n;i++){
        for(int j=1;j<=m;j++){
            dp[i][j] = max(dp[i][j-1],dp[i-1][j]);
            if(A[i] == B[j]) dp[i][j] = max(dp[i][j],dp[i-1][j-1]+1);
        }
    }
    cout<<dp[n][m]<<endl;
}

3.数字三角形

题目

给定一个如下图所示的数字三角形，从顶部出发，在每一结点可以选择移动至其左下方的结点或移动至其右下方的结点，一直走到底层，要求找出一条路径，使路径上的数字的和最大。

        7
      3   8
    8   1   0
  2   7   4   4
4   5   2   6   5

输入输出

输入

5
7
3 8
8 1 0 
2 7 4 4
4 5 2 6 5

输出

30

自底向上

思路

  1. 状态表示 dp[i][j] 表示从顶部走到dp[i][j] 路径上数字的和最大值
  2. 状态计算
  3. 初始化
  4. 目标值

代码

#include<iostream>
#include<algorithm>

using namespace std;

const int N=510;
int f[N][N],dp[N][N];

int main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
        for(int j=1;j<=i;j++){
            cin>>f[i][j];
        }
    }

    for(int i=n;i>=1;i--){
        for(int j=i; j>=1;j--){//正向反向都可以
            dp[i][j] = max(dp[i+1][j],dp[i+1][j+1])+f[i][j];
        }
    }
    cout<<dp[1][1]<<endl;
}

空间O(n)优化版
因为dp[i][j] 只与dp[i+1][j]和dp[i+1][j+1]有关，所以可以优化成一维数组


#include<iostream>
#include<algorithm>

using namespace std;

const int N=510;
int f[N][N],dp[N];

int main(){
    int n;
    cin>>n;
    for(int i=1;i<=n;i++){
        for(int j=1;j<=i;j++){
            cin>>f[i][j];
        }
    }

    for(int i=n;i>=1;i--){
        for(int j=1; j<=i;j++){
          //只能正向，如果反向的话，会导致计算前面的dp值时候，会被后面已经计算过的dp值污染，因为dp[j]由dp[j+1]转移而来
          //例如：正向循环，计算dp[1]后，再计算dp[2],不会用到dp[1],这样计算的结果就是正常的
          //而反向循环，先计算dp[2],然后再计算dp[1],而计算dp[1]又会用到dp[2]，这样dp[1]的值就被dp[2]污染，有重复加f[i][j]的值
            dp[j] = max(dp[j],dp[j+1])+f[i][j];
        }
    }
    cout<<dp[1]<<endl;
}

4 最大连续子序列和

题目

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

输入输出

输入: [-2,1,-3,4,-1,2,1,-5,4],
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

思路

这是是求连续子序列的最大和，所以转移方程为dp[i] = max(dp[i-1]+nums[i],nums[i]);
如果前面的和都没有当前数大，那么就以当前数为新的序列的起点

代码

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        if(!nums.size()) return 0;
        int n = nums.size();
        vector<int> dp(n);
        dp[0] = nums[0];
        int res = nums[0];
        for(int i=1;i<n;i++){
            dp[i] = max(dp[i-1]+nums[i],nums[i]);
            res = max(res,dp[i]);
        }
        return res;
    }
};

5 乘积最大连续子数组

题目

给你一个整数数组 nums ，请你找出数组中乘积最大的连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

输入输出

输入: [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。

思路

因为有负数，最大值 * 负数变为最小值，最小值 * 负数变为最大值。所以同时记录最大值和最小值，遇见负数时，两数交换。

代码

class Solution {
public:
    int maxProduct(vector<int>& nums) {
        if(!nums.size()) return 0;
        int n = nums.size();
        int max_res = nums[0],min_res = nums[0],res = nums[0];
        for(int i=1;i<n;i++){
            if(nums[i]<0) swap(max_res,min_res);
            max_res = max(max_res*nums[i],nums[i]);
            min_res = min(min_res*nums[i],nums[i]);
            res = max(res,max_res);
        }
        return res;
    }
};

6 鸡蛋掉落问题

题目描述

你将获得 K 个鸡蛋，并可以使用一栋从 1 到 N 共有 N 层楼的建筑。

每个蛋的功能都是一样的，如果一个蛋碎了，你就不能再把它掉下去。

你知道存在楼层 F ，满足 0 <= F <= N 任何从高于 F 的楼层落下的鸡蛋都会碎，从 F 楼层或比它低的楼层落下的鸡蛋都不会破。

每次移动，你可以取一个鸡蛋（如果你有完整的鸡蛋）并把它从任一楼层 X 扔下（满足 1 <= X <= N）。

你的目标是确切地知道 F 的值是多少。

无论 F 的初始值如何，你确定 F 的值的最小移动次数是多少？

输入输出

输入：K = 1, N = 2
输出：2
解释：
鸡蛋从 1 楼掉落。如果它碎了，我们肯定知道 F = 0 。
否则，鸡蛋从 2 楼掉落。如果它碎了，我们肯定知道 F = 1 。
如果它没碎，那么我们肯定知道 F = 2 。
因此，在最坏的情况下我们需要移动 2 次以确定 F 是多少。

思路

1 状态表示 dp[i][j] 表示 j个鸡蛋扔i次能测量的最长区间
2 状态计算 dp[i][j] = dp[i-1][j] + dp[i-1][j-1]+1

代码

class Solution {
public:
    int superEggDrop(int K, int N) {
        vector<vector<int>> dp(N+1,vector<int>(K+1,0));
        for(int i=1;i<=N;i++){
            for(int j=1;j<=K;j++){
                dp[i][j] = dp[i-1][j] + dp[i-1][j-1] +1;
            }
            if(dp[i][K]>=N){
                return i;
            }
        }
        return 0;
    }
};

7 俄罗斯套娃信封问题

问题描述

给定一些标记了宽度和高度的信封，宽度和高度以整数对形式 (w, h) 出现。当另一个信封的宽度和高度都比这个信封大的时候，这个信封就可以放进另一个信封里，如同俄罗斯套娃一样。

请计算最多能有多少个信封能组成一组“俄罗斯套娃”信封（即可以把一个信封放到另一个信封里面）。

输入输出

输入: envelopes = [[5,4],[6,4],[6,7],[2,3]]
输出: 3 
解释: 最多信封的个数为 3, 组合为: [2,3] => [5,4] => [6,7]。

思路

1.这是最长上升子序列问题的二维版，那么我们想办法将二维转为一维问题
2.首先按照宽度升序排序，这样后面信封的宽度一定>=前面的信封,然后将相等宽度的信封按照高度降序排列，
这里是为了排除掉宽度相等的情况。
3.对信封的高度取最长上升子序列就是答案。

代码

//两种方法实现，dp和二分
class Solution {
public:
    int maxEnvelopes(vector<vector<int>>& envelopes) {
        if(!envelopes.size()) return 0;
        sort(envelopes.begin(),envelopes.end(),[](const vector<int> &a,const vector<int> &b){
            if(a[0] == b[0]){
                return a[1]>b[1];
            }
            return a[0]<b[0];
        });
        int n = envelopes.size(),res = 0;
        //vector<int> dp(n,1);
        // for(int i=0;i<n;i++){
        //     for(int j=0;j<i;j++){
        //         if(envelopes[i][1]>envelopes[j][1]){
        //             dp[i] = max(dp[i],dp[j]+1);
        //         }
        //     }
        //     res = max(res,dp[i]);
        // }
        // return res;
        vector<int> dp(n,0);
        int cnt = 0;
        dp[cnt++] = envelopes[0][1];
        for(int i=1;i<n;i++){
            if(envelopes[i][1]>dp[cnt-1]) dp[cnt++] = envelopes[i][1];
            else{
                int l = 0,r = cnt-1;
                while(l<r){
                    int mid = (l+r)>>1;//[l,mid][mid+1,r]
                    if(dp[mid] >= envelopes[i][1]) r = mid;
                    else l = mid+1;
                }
                dp[l] = envelopes[i][1];
            }
        }
        return cnt;
    }
};
